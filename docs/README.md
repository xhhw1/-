## 核心结论

新建项目等同于新建一场对话。用户通过对话输入项目描述、上传资料、补充品牌信息；主 Agent 使用 DeepSeek-V4-Flash 判断下一步，调用资料解析、卖点提取、包装策略、详情页策略、出图、质检等分 Agent 和工具。每个关键结果通过对话卡片让人工确认，确认后写入有效项目记忆，并自动进入下一步。
